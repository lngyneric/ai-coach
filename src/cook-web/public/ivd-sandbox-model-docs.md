# IVD 沙盘推演 — 模型参数说明文档

> 版本: 1.0 | 更新: 2026-06

---

## 一、模型概述

沙盘推演使用 **多情景加权推演模型**，基于 S2（结构化调整）基准情景，通过调节 5 个外部参数和 3 个内部参数，生成 3-4 个不同概率情景的收入预测。

### 模型架构

```
输入层                       推导层                         输出层
┌──────────┐              ┌──────────────┐              ┌────────────┐
│ 用户目标   │              │  S1 保守防御  │ ← 下调因子   │  收入趋势   │
│ (自由文本) │              │  (概率25-35%) │              │  (堆叠图)   │
├──────────┤   ╱──→ 偏差 ├──────────────┤              ├────────────┤
│ 集采范围   │ ╱          │  S2 结构化调整 │ ← 基准模型   │  方案对比   │
│ 竞争强度   │ ────→     │  (概率40-50%) │ ← S&OP数据    │  (表格)     │
│ 渠道投入   │  ╲         ├──────────────┤              ├────────────┤
│ 技术投入   │   ╲──→    │  S3 主动进取  │ ← 上调因子   │  策略建议   │
│ 国产替代   │              │  (概率15-25%) │              │  (推荐)     │
├──────────┤              ├──────────────┤              ├────────────┤
│ 5个约束参数 │              │  S4 极限承压  │ ← 仅高压触发 │  推导日志   │
│ (1-5级)   │              │  (概率5-10%)  │              │  (可追溯)   │
└──────────┘              └──────────────┘              └────────────┘
```

## 二、参数定义与影响

### 2.1 参数总览

| 参数 | 类型 | 范围 | 默认值 | 对收入影响方向 |
|:----|:----|:---:|:-----:|:------------|
| 集采影响范围 | 分类 | 温和/中等/极限 | 中等 | 🔴 负面（降价→收入降） |
| 竞争强度 | 等级 | 1-5 | 3 | 🟡 中性偏负（价格战→利润降） |
| 渠道投入 | 等级 | 1-5 | 3 | 🟢 正面（更多覆盖→更多收入） |
| 技术投入 | 等级 | 1-5 | 3 | 🟢 正面（产品力→差异化溢价） |
| 国产替代压力 | 等级 | 1-5 | 3 | 🔴 负面（进口替代→份额流失） |

### 2.2 集采影响范围（分类参数）

| 取值 | 含义 | 基准收入调整 | 触发条件 |
|:---:|:----|:-----------:|:--------|
| `mild` | 温和 — 仅生化发光集采 | ×0.95 | 假设政策温和推进 |
| `moderate` | 中 — 凝血2027入集采 | ×0.90 | 默认最可能情景 |
| `severe` | 极限 — 血液+凝血全部集采 | ×0.80 | 压力测试情景 |

**影响路径**：

```
集采范围 → 基准收入系数(s1Adj/s2Adj) → 所有产品线收入等比调整
```

**公式**：
```
if 集采 = mild:   baseAdj = 0.95
if 集采 = moderate: baseAdj = 0.90
if 集采 = severe:  baseAdj = 0.80

各方案收入_i = S2基准值 × baseAdj × 其他参数乘积
```

### 2.3 竞争强度（1-5 级）

| 等级 | 标签 | 描述 | 对策略影响 |
|:---:|:----|:----|:---------|
| 1 | 极低 | 各玩家和平共处 | 可提价，利润率上升 |
| 2 | 低 | 局部市场竞争 | 选择性降价 |
| 3 | 中 | 主流市场竞争 | 维持现价（基准） |
| 4 | 高 | 价格战局部爆发 | 被迫跟进降价 |
| 5 | 极高 | 全面价格战 | 全线降价保份额 |

**影响路径**：不直接修改收入公式，而是影响**可信度评估**和**策略建议**。高竞争强度下，激进方案(S3)可信度降低，保守方案(S1)可信度升高。

### 2.4 渠道投入（1-5 级）

| 等级 | 标签 | 渠道乘数 | 描述 |
|:---:|:----|:--------:|:----|
| 1 | 放弃 | ×0.90 | 减少销售团队，仅维持现有客户 |
| 2 | 收缩 | ×0.95 | 优化编制，聚焦高价值客户 |
| 3 | 维持 | ×1.00 | 保持现有渠道投入（基准） |
| 4 | 扩张 | ×1.04 | 扩编销售团队，增加经销商覆盖 |
| 5 | 全力 | ×1.08 | 全渠道扩张，平台商+直销双线推进 |

**公式**：
```
渠道乘数 = 1 + (channelLevel - 3) × 0.04

示例:
  channel=4 → 乘数 = 1 + 1×0.04 = 1.04 (+4%)
  channel=2 → 乘数 = 1 + (-1)×0.04 = 0.96 (-4%)
```

### 2.5 技术投入（1-5 级）

| 等级 | 标签 | 技术乘数 | 描述 |
|:---:|:----|:--------:|:----|
| 1 | 暂停 | ×0.93 | 冻结研发，仅维护现有产品 |
| 2 | 缩减 | ×0.965 | 减少研发预算，聚焦短周期项目 |
| 3 | 维持 | ×1.00 | 保持现有技术投入（基准） |
| 4 | 加强 | ×1.03 | 增加研发，加快新产品上市 |
| 5 | 领先 | ×1.06 | 全力投入技术领先，建立壁垒 |

**公式**：
```
技术乘数 = 1 + (techLevel - 3) × 0.03

示例:
  tech=5 → 乘数 = 1 + 2×0.03 = 1.06 (+6%)
  tech=1 → 乘数 = 1 + (-2)×0.03 = 0.94 (-6%)
```

### 2.6 国产替代压力（1-5 级）

| 等级 | 标签 | 描述 | 对策略影响 |
|:---:|:----|:----|:---------|
| 1 | 无 | 进口产品不受限 | — |
| 2 | 低 | 部分三级医院有国产配额 | 轻微影响高端市场 |
| 3 | 中 | 国产替代政策持续推进 | 基准情景 |
| 4 | 高 | 三级医院国产配额>50% | 显著影响，触发S4 |
| 5 | 极高 | 进口替代限期完成 | 极限压力，触发S4 |

**影响路径**：
- 国产替代 ≥ 4 时，自动生成 S4（极限承压）情景
- 影响免疫(HISCL)和高端血液产品线的可信度预期

## 三、情景推导算法

### 3.1 收入计算

```
情景_i_产品线_j_年_k 收入 = 
  S2基准值[j][k] × 
  集采调整(baseAdj) × 
  渠道乘数(channelFactor) × 
  技术乘数(techFactor) × 
  年度趋势(annualTrend)
```

### 3.2 各情景公式

#### S1 保守防御

```
收入 = S2基准 × baseAdj × (0.85 + 年序×0.03)
     = S2基准 × baseAdj × [0.85, 0.88, 0.91, 0.94]

推导逻辑: 集采冲击→下调基准→保守运营→逐年小幅恢复
可信度: baseAdj≥0.90时高(≥70%), 否则中
```

#### S2 结构化调整（基准）

```
收入 = S2基准 × baseAdj × channelFactor × techFactor

推导逻辑: S&OP基准数据 → 集采调整 → 渠道/技术乘数
可信度: 偏差最小路径 → 最高可信度(75%)
```

#### S3 主动进取

```
收入 = S2基准 × min(1.3, s3Adj + 年序×0.05)

其中 s3Adj = 1 + (channel-3)×0.04 + (tech-3)×0.03

推导逻辑: 渠道+技术双投入 → 逐年递加(+5%/年) → 上限+30%
可信度: 需要渠道和技术同时有利 → 综合概率低(45%)
```

#### S4 极限承压（仅高压触发）

```
收入 = S2基准 × 0.75 × (1 - 年序×0.05)
     = S2基准 × [0.75, 0.7125, 0.675, 0.6375]

触发条件: 国产替代≥4 或 集采=severe
推导逻辑: 全面冲击 → 逐年恶化(-5%/年)
可信度: 边界场景 → 低(25%)
```

### 3.3 多产品线汇总

```
总收入_年 = Σ(各产品线收入_年)

堆叠柱状图展示:
  横轴: 2024A ~ 2028E (5年)
  纵轴: 收入(基准100)
  堆叠: 按产品线拆分
```

## 四、可信度评估标准

### 4.1 评分规则

| 可信度 | 分数 | 判定标准 |
|:-----|:---:|:---------|
| 🟢 高 | ≥70% | 与 S2 基准偏差 ≤ 20%，且参数在历史波动范围内 |
| 🟡 中 | 45-70% | 需要 1-2 个非基准假设同时成立 |
| 🔴 低 | <40% | 需要多个小概率事件同时发生 |

### 4.2 各情景可信度

| 情景 | 通常可信度 | 判定依据 |
|:----|:---------:|:---------|
| S1 保守防御 | 65-75% | baseAdj 接近 1.0 时高，否则中 |
| S2 结构化调整 | 65-75% | 默认基准，偏差最小 |
| S3 主动进取 | ~45% | 需要渠道+技术双有利，概率中 |
| S4 极限承压 | ~25% | 需要集采+国产替代双重极限，概率低 |

### 4.3 概率分配

```
S1 概率 = 25-35%  (保守)
S2 概率 = 40-50%  (基准，最可能)
S3 概率 = 15-25%  (激进)
S4 概率 = 5-10%   (极限，仅高压时)
```

### 5.2 华鑫渠道模型（新增）

华鑫渠道在 S2 基准中呈缓慢下降趋势（-3.1% CAGR），反映：
- 华鑫单一代理模式增量有限
- XS 系列逐步退市影响
- XNL 替代 XS 部分对冲

**培训赋能效应**（2026.06 华鑫产品线培训课程上线）：
```
假设培训提升经销商能力 → 渠道效率提升
S1: 培训维持现有授权 → 下降放缓
S2: 培训+考核 → 降幅收窄
S3: 培训+扩张 → 渠道保持稳定
```

## 七、使用约束

### 7.1 模型限制

1. **线性假设**：当前模型使用线性乘积，未考虑非线性效应（如规模效应、网络效应）
2. **独立假设**：各产品线独立计算，未考虑产品线间交叉影响（如流水线方案锁定效应）
3. **静态弹性**：参数弹性系数固定（渠道±4%/级、技术±3%/级），实际弹性可能随环境变化
4. **无时间序列自相关**：各年独立调整，未考虑上年结果的反馈效应

### 7.2 最佳实践

1. **多次推演**：建议调整 1-2 个参数后重新推演，对比结果变化
2. **关注 S2+S3 区间**：最可能结果落在基准和激进情景之间
3. **压力测试**：设置极端参数（集采=极限+竞争=5+国产=5）验证抗压能力
4. **参考历史偏差**：2024A 数据为实际值，2025E 为预估值，关注预测偏差率
