# 一线培训带教流程设计

## 课程架构

```
一线培训带教流程（总课程）
├── Phase 1: 入职培训带教（入职日 — 3天内）
│   ├── 公司制度（HR 督导）
│   ├── 培训准备（上司督导）
│   └── 工作流程（助理团队督导）
│
├── Phase 2: 「三个明白」带教
│   ├── 第一阶段（1个月）
│   │   ├── 产品理论知识培训
│   │   ├── 市场政策学习
│   │   ├── 任务数字确认
│   │   └── 现场带教
│   │
│   ├── 第二阶段（1个月）
│   │   ├── 产品理论知识培训（免疫/生化/糖化/Alifax）
│   │   ├── 市场政策学习
│   │   ├── 任务数字确认
│   │   └── 现场带教
│   │
│   └── 第三阶段（1个月）
│       ├── 产品理论知识培训（生科/招标）
│       └── 现场带教（压货/招标）
│
└── Phase 3: 专业知识自学（在线 e-learning，3个月）
    ├── 血液检验基础
    ├── 尿液检验基础
    ├── 凝血检验基础
    ├── 免疫检验基础
    ├── 生化检验基础
    └── 生科检验基础
```

## 阶段验收标准

### Phase 1 验收
- [ ] 入职手续完成
- [ ] 员工手册签署
- [ ] MBO 初稿完成
- [ ] 一周带教计划制定

### Phase 2 验收（三个明白）
- [ ] 产品知识考核通过
- [ ] 市场政策理解确认
- [ ] 任务数字达成路径清晰
- [ ] 现场带教记录完整

### Phase 3 验收
- [ ] 在线 e-learning 完成率 ≥ 80%
- [ ] 基础知识考核通过

## 合规控制点
1. 每个阶段需带教导师签字确认
2. 学习记录自动同步到培训档案
3. 阶段验收不通过需制定改进计划
