## 型号命名规则

| 后缀 | 含义 |
|------|------|
| `C` / `_C` | China 中国本土化定制版本，国内医院主流装机 |
| `i` | iterate 迭代优化款，硬件与软件升级 |
| `PRO` | Professional 专业升级款 |
| `Series` | 系列化产品家族 |

## 国产化进程

共 13 款国产化型号，覆盖血液/凝血/尿液/免疫/生科/系统全产品线。
国产化策略: 进口仪器原版 + 国产C版双线并行，C版适配国内集采政策与价格体系。
