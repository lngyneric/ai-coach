## 一、Nginx 配置说明

### 端口

| 端口 | 协议 | 用途 |
|:----:|:----:|------|
| 80 | HTTP (内网重定向) | 自动重定向到 8080 |
| 8080 | HTTP (全功能) | 内网访问，含所有API和前端 |
| 443 | HTTPS (外网) | SSL 加密外网访问，主域名 `eu.sysmex.com.cn` |

### 容器架构

```
用户请求 (浏览器)
     │
     ▼
  ai-shifu-nginx  (端口 80/8080/443)
     │
     ├── 静态文件（HTML/JS/CSS） → 直接返回
     ├── /api/*                 → ai-shifu-api:5800
     ├── /api/chat/             → api.deepseek.com (代理)
     ├── /c/                    → ai-shifu-cook-web:5000
     ├── /                      → ai-shifu-cook-web:5000
     ├── /fastclaw/             → FastClaw Gateway:18953
     ├── /courses/              → 课程服务 :8802
     └── /training              → 培训服务 :8803
```

### 配置位置

```
宿主机: /home/sysmex/ai-shifu/docker/nginx.conf
容器内: /etc/nginx/nginx.conf (通过 bind mount 同步)
```

修改宿主机文件后：

```bash
docker restart ai-shifu-nginx
```

## 六、部署维护命令

### 热部署静态文件

```bash
# 更新 IVD 知识图谱
docker cp src/cook-web/public/ivd-knowledge-tree.html ai-shifu-nginx:/usr/share/nginx/html/

# 更新个人能力图谱
docker cp src/cook-web/public/personal-growth-tree.html ai-shifu-nginx:/usr/share/nginx/html/

# 更新登录页
docker cp docker/sysmex-login.html ai-shifu-nginx:/usr/share/nginx/html/
```

### 配置变更

```bash
# 修改 nginx 配置后重启
docker restart ai-shifu-nginx

# 验证配置
docker exec ai-shifu-nginx nginx -t
```

### 容器管理

```bash
# 查看状态
docker ps

# 查看日志
docker logs ai-shifu-nginx --tail 50
```

### 数据库备份

```bash
# 手动触发备份
bash /home/sysmex/ai-shifu/scripts/db_backup.sh

# 查看备份文件
ls -lh /home/sysmex/ai-shifu/docker/backups/

# 查看备份日志
cat /home/sysmex/ai-shifu/docker/backups/backup.log
```

备份策略：
- ⏰ **定时**: 每天北京时间 01:00，由 systemd timer 自动触发
- 📦 **格式**: mysqldump → gzip 压缩（~4.4MB/次）
- 🗑️ **保留**: 30 天自动清理
- 🛠️ **方式**: `docker exec ai-shifu-mysql mysqldump` 在容器内执行

查看定时器状态：
```bash
systemctl --user list-timers ai-shifu-db-backup.timer
systemctl --user status ai-shifu-db-backup.timer
```
