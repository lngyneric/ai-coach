## 一、平台总览

### 1.1 平台是什么

Sysmex 培训平台是一个**知识图谱驱动的 AI 学习平台**，将散落的课程组织成结构化的知识网络，让学习不再是孤立的"刷课"，而是沿着知识脉络系统成长。

### 1.2 核心能力矩阵

| 能力 | 说明 | 适用场景 |
|:----|:-----|:---------|
| 🧬 **知识图谱** | 课程按知识领域组织成节点网络 | 探索式学习、查漏补缺 |
| 💬 **AI 对话** | 每个知识点可触发 AI 对话答疑 | 即问即答、深度理解 |
| 📚 **课程体系** | 从图谱节点跳转到完整课件 | 系统学习、考试认证 |
| 📋 **带教计划** | 分阶段的学分制学习路径 | 新人入职培训、岗位晋升 |
| 🏰 **沙盘推演** | IVD 销售策略模拟推演 | 销售培训、战略规划 |
| 🔍 **知识缺口** | 红色节点表示待开发内容 | 课程规划、需求反馈 |

### 1.3 双端适配策略

| 维度 | PC / 大屏 | 手机 / 平板 |
|:----|:----------|:-----------|
| **推荐入口** | 🧬 **IVD 知识图谱** | 📱 **学习首页** |
| **核心理由** | 图谱需要大屏幕拖拽/缩放 | 首页列表式布局适配小屏 |
| **交互方式** | 鼠标拖拽、悬停、点击 | 上下滑动、点击 |
| **课程学习** | 同 `/c/{id}` 课件页 | 同 `/c/{id}` 课件页 |

## 二、首次使用引导（7 步走）

### 🚶 第一步：选择入口

**PC 用户（推荐）**
```
浏览器打开 → https://eu.sysmex.com.cn/ivd-knowledge-tree.html
```

**手机用户（推荐）**
```
浏览器打开 → https://eu.sysmex.com.cn/
```

> **为什么这样推荐？**
> - 知识图谱是**力导向图**，节点拖动/缩放在大屏幕上体验更好
> - 手机端首页是**列表式布局**，上下滑动浏览更自然
> - 两个入口最终都通向同一个课程体系

**基本操作：**

| 操作 | 效果 |
|:----|:------|
| 🖱️ 拖拽空白区域 | 平移画布 |
| 🔄 滚轮 | 缩放图谱 |
| 👆 点击节点（圆圈） | 打开 AI 对话窗 |
| 🖱️ 悬停节点 | 显示详情 + 课程链接 |
| 🔍 点击右上角「缺口」 | 高亮无课程的红色节点 |
| 🔄 点击「重置」 | 回到初始视图 |

**节点颜色含义：**
- 🟢 **绿色实心** = 该知识点已有配套培训课程
- 🔴 **红色虚线** = 课程缺口（待开发，欢迎反馈需求）
- 🟠 **橙色虚线** = 优先创建的课程

**连线含义：**
- 🔵 实线 = 前置知识（建议先学这些）
- 🟣 实线 = 延伸内容（学完这个可以继续学）
- 🟢 虚线 = 并行内容（可以同时学习）

### 🚶 第四步：AI 对话学习

点击任意节点 → 右下角弹出对话窗

```
┌──────────────────────────────────┐
│ 💬 AI 学习助手                    │
│                                  │
│ 你好！我是检验医学AI助手。         │
│ 你正在学习「血液学基础」，         │
│ 有什么可以帮助你的？              │
│                                  │
│ ┌──────────────────────────┐     │
│ │ 输入问题...         [发送]│     │
│ └──────────────────────────┘     │
└──────────────────────────────────┘
```

**可以问什么：**

| 问题类型 | 示例 |
|:--------|:-----|
| 概念解释 | "XN-9000 和 XN-1000 有什么区别？" |
| 临床应用 | "APTT 延长常见于哪些疾病？" |
| 操作指导 | "UF-5000 报 E1001 错误怎么处理？" |
| 学习路径 | "我是新招的技术，应该先学什么？" |
| 对比分析 | "HISCL 和罗氏 e801 哪个更好？" |

**对话特性：**
- ✅ 回复**逐字显示**（流式输出，不用等全部生成）
- ✅ AI **知道你正在看哪个知识点**（自动加载课程大纲）
- ✅ 每次回答 200 字以内，**专业且易懂**
- ✅ 关闭对话窗后，右下角保留 💬 按钮可重新打开

## 四、常见问题与注意事项

### 4.1 网络与访问

| 问题 | 说明 |
|:----|:-----|
| 🌐 **外网访问** | `https://eu.sysmex.com.cn/` 全部功能可用 |
| 🔒 **HTTPS** | 生产环境强制 HTTPS，请勿使用 HTTP 访问 |
| 🚀 **加载速度** | 首次加载约 2-5 秒（受图谱数据量影响） |
| 🔄 **刷新** | 如遇页面卡死，刷新即可恢复 |
| ⚠️ **AI 回复超时** | 如 AI 回复 "(未收到回复)"，刷新重试 |

### 4.2 功能限制

| 功能 | 限制 | 建议 |
|:----|:-----|:-----|
| 🧬 知识图谱 | 仅 PC 大屏最佳体验 | 手机可查看但操作不便 |
| 💬 AI 对话 | 每次 200 字以内 | 可多次提问深入探讨 |
| 📚 课程课件 | 部分需登录/付费 | 免费课程可直接学习 |
| 🏰 沙盘推演 | 仅 IVD 方向 | 销售培训专用 |
| 🔴 红色节点 | 暂无课程，待开发 | 可反馈课程需求 |

### 4.3 账号与权限

- **未登录用户**：可浏览图谱、查看部分免费课程
- **已登录用户**：学习进度跟踪、付费课程购买
- **管理员**：课程管理、用户管理、数据统计

## 六、快速问题排查

| 现象 | 可能原因 | 解决方法 |
|:----|:---------|:---------|
| 图谱空白/不显示 | 网络问题或 D3.js 加载失败 | 刷新页面 |
| AI 对话不回复 | 网络超时或 DeepSeek 接口问题 | 刷新后重试 |
| 课程链接点不了 | 需先悬停节点再移动鼠标到浮窗 | 鼠标移到浮窗再点击 |
| 对话窗打不开 | 未点击节点 | 点击任意节点触发 |
| 手机端图谱太挤 | 知识图谱不适配小屏 | 改用首页浏览 |
| 页面显示404 | 路由未配置 | 检查 URL 是否正确 |
| 图片/样式丢失 | CDN 资源加载失败 | 刷新或清除缓存 |
